# Arquitectura del Sistema

## Stack Tecnológico

-   Backend: Laravel 12 o superior (API REST)
-   Frontend: React + Vite + Tailwind CSS
-   Base de datos: MariaDB
-   Contenido: Archivos JSON versionados
-   Gráficas: Chart.js

## Arquitectura

Cliente React -\> API Laravel -\> Servicios -\> MariaDB/JSON

## Módulos

-   Administración
-   Gestión académica
-   Contenido
-   Banco de preguntas
-   Sesiones QR
-   Evaluaciones
-   Reportes
-   Dashboard

## Principios

-   API First
-   Modular
-   Escalable
-   Reutilizable
-   Separación entre contenido y lógica
